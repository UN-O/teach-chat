# 技巧十八：截圖風險意識與用字自審

> 適用情境：學生打架｜難度：中｜優先練習：★★

---

## 那張截圖

下午四點半，打架事件剛處理完，老師在疲憊與焦慮中拿起手機，對打架學生的家長傳了一句話：

「你家孩子真的太過分了，這樣下去我沒辦法保證他不會再犯。」

她的本意是表達教育的挫折感，也想讓家長正視問題的嚴重性。但她沒有預料到的是，這句話在三十分鐘內被截圖，轉傳進了班上的家長群組。

到了隔天早上，校長室收到四通投訴電話。指控的內容是：「老師公開指責孩子人格，還暗示教育放棄。」

老師目瞪口呆。她說的是事實，卻變成了呈堂的證據。

---

## 數位環境的兩個現實

理解截圖風險，必須先接受數位環境的兩個根本特性：

### 永久性（Permanence）

在 LINE 上說出的話，不會因為時間流逝而消失。對方的手機裡永遠存著那則訊息。即使你刪掉自己那邊的紀錄，對方手機中的版本依然完整。一週後、一學期後、一年後，當親師關係因為其他原因惡化，那則訊息都可能被重新翻出，以全新的敵對角度重新詮釋。

### 傳播性（Virality）

一則截圖從私訊傳到家長群組，只需要兩秒。從群組流出到社群媒體，再被陌生人討論，也不過幾個小時。截圖會跨越你原本預期的對話範圍，到達你完全不認識的人手上。更重要的是，截圖是靜態片段——它剝除了所有前後文、語境與關係脈絡，只留下那一句「最刺眼」的話。

根據學術期刊《Journal of Computer-Mediated Communication》的研究，截圖在通訊平台上的蒐集與分享行為，在技術層面完全不留任何痕跡，傳送者無從得知截圖是否發生。這意味著每一次傳訊，你都必須預設對方手機的快門已經按下。

---

## 截圖作為投訴證據的現實

台灣的教育申訴實務中，LINE 訊息截圖早已成為家長向學校投訴的第一份佐證。只要截圖能顯示傳送者身份與時間戳記，學校行政單位通常就會受理，啟動調查程序。

打架糾紛尤其高風險。原因在於：雙方家長情緒都處於高點，訊息往來密集，且每一方都有強烈的動機找到「對方有錯」的佐證。如果你在傳訊時帶著情緒，不假思索地使用了評斷性、承諾性或預測性的語言，那些字句就會成為完美的截圖素材。

「你家孩子一直是問題學生」——貼標籤，可能構成對未成年人名譽的不當陳述。

「我保證這件事不會再發生」——承諾，一旦再度發生，就成了你「失職」的證據。

「這樣下去一定會有後果」——情緒化預測，被解讀為威脅或恐嚇。

## 截圖測試：一個養成自審習慣的方法

在打字完畢、按下傳送之前，停頓三秒，心裡默問：

**「這句話被截圖後，在家長群組裡，陌生人讀到它，會怎麼解讀？」**

這個動作稱為「截圖測試」（Screenshot Test）。它不是要你變得過度謹慎、無法溝通，而是讓你在衝動傳訊與深思傳訊之間插入一道緩衝。

截圖測試關注三個問題：

1. **這句話脫離上下文後，還成立嗎？** 若只看這一句，意思是否會完全翻轉？
2. **這句話裡有沒有我無法兌現的承諾？** 承諾是截圖陷阱的重災區。
3. **這句話是在描述事實，還是在表達情緒？** 事實可以被驗證，情緒容易被詮釋成攻擊。

## 高風險用語辨識與替換清單

以下是打架處理情境中，最常出現的高截圖風險語句，以及低風險的對應替換版本：

### 評斷人格

| 高截圖風險 | 低截圖風險 |
|------------|------------|
| 你家孩子真的太過分了 | 今天發生的行為需要我們一起關注 |
| 他這樣的行為很不正常 | 這個行為在學校規範中屬於需要輔導的範疇 |
| 這孩子根本不知道尊重 | 目前觀察到他在人際互動上需要支持 |

### 情緒化承諾

| 高截圖風險 | 低截圖風險 |
|------------|------------|
| 我保證不會再讓這種事發生 | 學校會持續關注並採取相應的輔導措施 |
| 這件事我一定會處理好 | 我們已啟動學校的標準處理程序，會持續追蹤 |
| 你放心，我替你盯著他 | 我會定期與您更新情況 |

### 預測與威脅語氣

| 高截圖風險 | 低截圖風險 |
|------------|------------|
| 這樣下去一定會出問題 | 目前的情況需要及早介入，避免後續影響 |
| 他再這樣下去我沒辦法 | 如果情況持續，我們可能需要討論進一步的輔導資源 |
| 你不管，後果自負 | 家長的積極參與對孩子的改變非常重要，我們希望一起合作 |

### 猜測與未確認事實

| 高截圖風險 | 低截圖風險 |
|------------|------------|
| 應該是你家孩子先動手的 | 目前我們還在釐清事發經過，會在確認後告知您 |
| 他大概是故意的 | 我們需要進一步了解當時的情況 |
| 我猜對方家長也不知道 | 我會分別聯繫雙方家長，確保資訊一致 |

## 三個實作練習

**練習一：「傳送前三秒」**
設定一個習慣：每次按下傳送前，強迫自己重讀一遍訊息，想像陌生家長看到這句話的反應。初期可以在手機桌布寫上提示語，提醒自己不要衝動。

**練習二：「情緒冷卻視窗」**
打架事件處理完的當下，不要立刻傳訊息。給自己至少三十分鐘的冷卻時間，讓腎上腺素退潮後再拿起手機。實務上，可以先將草稿訊息存在備忘錄，等情緒平穩後再修改傳送。

**練習三：「事實與感受分離」**
在傳訊息前，問自己：「這句話是在陳述可驗證的事實，還是在表達我此刻的感受？」感受不必壓抑，但不適合出現在 LINE 訊息中。感受屬於面對面談話的素材，不屬於數位紀錄。

## 一個關鍵觀念

截圖測試不是要你成為一個沒有溫度的機器人，而是要你成為一個在數位環境中有智慧的溝通者。

有溫度的訊息可以這樣寫：「我知道今天的消息對您來說很突然，也很難受。我們一起來想辦法。」這句話充滿關懷，又完全通過截圖測試。

專業不等於冷漠。它意味著你在表達關心的同時，依然守住了讓自己免於誤解與攻擊的語言邊界。
